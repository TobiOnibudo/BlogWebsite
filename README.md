# Blog Website
Designed Blog Web app where users could post blog about anything on their mind.

## Tech stack
This project utilizes the following languages, frameworks and vcs
+ JavaScript
+ Node JS 
+ Mongo DB
+ Express
+ EJS 
+ CSS 
+ HTML
